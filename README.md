# 🏦 Indian Retail Loan Approval Analytics Dashboard

## 📌 Project Overview

The **Indian Retail Loan Approval Analytics Dashboard** is an interactive Power BI project designed to analyze loan applications, approval trends, customer behavior, and key factors influencing loan approvals.

This dashboard enables banking professionals, credit analysts, and decision-makers to monitor loan performance, understand customer segments, and improve lending strategies through data-driven insights.

---

## 🎯 Business Objective

The primary objectives of this project are:

- Analyze loan approval and rejection trends
- Understand customer behavior patterns
- Evaluate the impact of income, employment, and CIBIL score on approvals
- Identify high-performing customer segments
- Support data-driven lending decisions

# 📈 Dashboard Pages

## 1️⃣ Executive Summary Dashboard

### Features

- Total Applications
- Approved Loans
- Rejected Loans
- Approval Rate
- Total Loan Amount Requested
- Monthly Loan Application Trend
- Loan Purpose Distribution
- Application Distribution by Customer Segment

### Key Insights

- Over **9,000 loan applications** were received.
- The overall **approval rate is 56%**.
- Personal Loans account for the highest number of applications.
- Loan applications peaked during **May**.
- Total requested loan amount exceeded **₹8 Billion**.

---

## 2️⃣ Customer Behaviour Analysis Dashboard

### Features

- Loan Approvals by Gender
- Employment Type Analysis
- CIBIL Score Impact Analysis
- Annual Income Analysis
- Residence Type Analysis
- City Tier Distribution

### Key Insights

- Male applicants submitted the highest number of applications.
- Salaried private employees represent the largest applicant group.
- High and Medium CIBIL scores significantly improve approval chances.
- High-income customers receive more loan approvals.
- Tier-1 and Tier-2 cities contribute the majority of approved loans.
